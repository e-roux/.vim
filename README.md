# vim dotfiles

This is my `.vim` 

## Install

```
git clone git@github.com:fesaille/.vim.git ~/.vim
echo 'runtime vimrc' >> ~/.vimrc
cd ~/.vim && git submodule update --init --recursive
```

## Usage

### Plugin: Ack

```
:Ack [options] {pattern} [{directories}]
```
